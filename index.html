Arch Install Guide

Section 1.0: Network

Before continuing throught this setup check that you have to do this.

$ ping 1.1.1.1

If you get something along the lines off

$ 64 bytes from 1.1.1.1: icmp_seq=1 ttl=57 time=18.7 ms

then you have a working network connection and should scroll to Section 2.0.

If not no worry's this is a simple fix in the install procces.

Section 1.1: Setting up a temporary connection

First off run this following command

$ iwctl

This will enter into iwctl. Next run.

$ device list

This will display your wifi interface's. Next run.

$ station (interface) scan

This will start a scan for networks. Next run.

$ station (interface) scan

Now to get the names of availbe networks run the command below.

$ station (interface) get-networks

Now it will prompt you for a keyphrase enter your wifi password.

This will connect to the network you want to connect to. Next run.

$ exit

That will exit iwctl

Section 1.2: Testing Connection

$ ping 1.1.1.1

If you get something along the lines off

$ 64 bytes from 1.1.1.1: icmp_seq=1 ttl=57 time=18.7 ms

then you have a working network connection and should scroll to Section 2.0 if not go back to 1.1.

Section 2.0: Setting up the system

Now that you have a working network connection you can start setting up the system.

First off run this following command

$ archinstall

This will enter into archinstall.

Now you should be shown a list of numbers and intials of countrys.

Now to select a country enter the number that matches your country.

Example for the UK.

$ 25

Now you should be shown a list of numbers and languages.

Now to select a language enter the number that matches your language.

Example for the UK.

$ 54

Now you should be shown a list of numbers and diffrent drives.

Now to select a drive enter the number that matches your drive.

Example for my Computer(sda).

$ 1
